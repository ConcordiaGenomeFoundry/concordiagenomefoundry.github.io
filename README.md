# Concordia Genome Foundry Website

# =========================
## Description
### Website for the Concordia Genome Foundry, a synthetic biology lab at Concordia University in Montreal, Canada.
### This website is built using Jekyll and hosted on GitHub Pages.

## Installation
### To run the website locally, you need to have Ruby and Jekyll installed.

### 1. Install Ruby: https://www.ruby-lang.org/en/documentation/installation/
### 2. Install Jekyll: https://jekyllrb.com/docs/installation/
### 3. Run the following commands in your terminal:

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```
